# web_personal
## Descripción
Este proyecto es una página web personal creada con HTML. Incluye secciones como Presentación, Sobre mi, Habilidades y Contactos, con enlaces internos y una imagen en formato `figure`.

## Estructura del proyecto
- `index.html`: página principal
- `assets/`: imágenes y recursos utilizados

## Cómo visualizar el proyecto
1. Abrí el archivo `index.html` en tu navegador.
2. Si corresponde, asegurate de que la carpeta `assets/` esté en la misma ubicación que el `index.html`.

## Tecnologías
- HTML
